Welcome to Windhill!

Windhill is a episodic adventure game with an interactive storyline based on a fantasy world, with main characters Max and Chloe. This game is brought to you by Xpark Games and 6TFPS Studio.

How to Play Windhill?

You need a laptop, computer or a tablet to play this game. The game contains puzzles which requires intelligence, patience and time to solve. With each succes you will be elevate to the following storyline.

Rules

The entire game is based on solving one big mystery, each episode in the end reveals something new which opens more about the mystery.
